# 侵入组件问题

  • 加入scoped, 入侵组件class时会失败；

  • ::v-deep

<style lang="scss" scoped>
   ::v-deep .van-sticky{
     background-color: #ffffff;
   }
</style>
  • 新增一个style无scoped标签控制组件样式
/*样式入侵*/
<style>
.tabs .van-sticky{
  overflow: hidden;
  border-radius: 22px;
}
</style>
  • CSS Modules 中使用 :global 伪类来定义全局样式
/* styles.module.css */
.localClass {
    color: red;
}

:global(.globalClass) {
    color: blue;
}
